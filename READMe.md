# Implementación de QuickSort en Java

## 📌 Descripción
Este proyecto implementa el algoritmo de ordenamiento QuickSort en Java utilizando el enfoque de divide y vencerás.

## ⚙️ Funcionamiento
El algoritmo selecciona un pivote y reorganiza el arreglo de forma que:
- Los elementos menores quedan a la izquierda
- Los mayores a la derecha

Luego aplica recursividad.

## 🧮 Complejidad
- Mejor caso: O(n log n)
- Caso promedio: O(n log n)
- Peor caso: O(n^2)

## ▶️ Ejecución
1. Compilar:
```
javac QuickSort.java
```

2. Ejecutar:
```
java QuickSort
```

## 📊 Ejemplo
Entrada:
```
8 3 1 7 0 10 2
```

Salida:
```
0 1 2 3 7 8 10
```
